== Description ==

AI Category Content Generator automatically creates posts under your chosen categories using AI-generated content. Designed for bloggers, affiliate marketers, and content creators looking to streamline their publishing process.

= Features =
* Automatically generate posts for selected categories.
* Schedule content generation daily, hourly, or twice daily using WordPress cron.
* Easy-to-use admin settings interface.
* Posts are saved as standard WordPress posts for easy editing.
* Lightweight and developer-friendly.

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/ai-category-content-generator directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress.
3. Navigate to Settings → AI Category Content Generator.
4. Choose your categories and save settings.

== Usage ==

Once configured, the plugin will generate AI-powered posts under the selected categories on the defined schedule.

To use:

1. Go to WP Admin → Settings → AI Category Content Generator.
2. Select the categories you want content for.
3. Choose your generation schedule.
4. Save and let it run in the background.

Posts will appear under Posts → All Posts automatically.

== Frequently Asked Questions ==

= Does it modify existing posts? =
No. It only creates new posts and does not alter your existing content.

= How does it generate content? =
It uses predefined logic or an AI API (e.g., OpenAI) to craft new content based on your selected categories.

= Can I edit the generated posts? =
Yes, like any post in WordPress.

= Can I use it with custom post types? =
Currently, it supports default post types only.

= Will it work with Classic Editor and Gutenberg? =
Yes, generated posts are fully compatible.
